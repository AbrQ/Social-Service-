1. La integral de convolución.



La convolución de dos señales x(t) y h(t) es una nueva función de tiempo y(t):

y(t) = h(t) * x(t) = x(t) * h(t)


Propiedades



1. h(t)*x(t) = x(t)*h(t)

2. h(t)*[ax(t)] = a[h(t)*x(t)] donde a es una constante

3. h(t)*[x1(t) + x2(t)] = h(t)*x1(t) + h(t)*x2(t)

4. h(t)*[x1(t)*x2(t)] = [h(t)*x1(t)]*x2(t)

5. Si h(t) está limitada en tiempo a (a,b) y x(t) está limitada a (c, d), entonces h(t)*x(t) está limitada en tiempo a(a+c, b+d).

6. Si A1 es el área bajo h(t) y A2 es el área bajo x(t), entonces el área bajo h(t)*x(t) es A1A2.
